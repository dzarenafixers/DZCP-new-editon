كلاس Player

يمثل هذا الكائن لاعبًا متصلًا بالسيرفر ويوفر جميع الخصائص والوظائف المتعلقة به.

الخصائص الرئيسية

الخاصية النوع الوصف
UserId string المعرف الفريد للاعب
Nickname string اسم اللاعب في اللعبة
Health float نقاط الصحة الحالية (0-100)

الوظائف الأساسية

SendMessage

إرسال رسالة للاعب

player.SendMessage("مرحباً بك في السيرفر!");

Teleport

نقل اللاعب إلى موقع معين

player.Teleport(new Vector3(10, 5, 20));
