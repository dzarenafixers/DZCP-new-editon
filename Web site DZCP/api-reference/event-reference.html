مرجع أحداث DZCP

أحداث اللاعب

OnPlayerJoin

يتم استدعاؤه عندما ينضم لاعب جديد للسيرفر

EventManager.OnPlayerJoin += player =>
{
    player.SendMessage("مرحباً بك في السيرفر!");
};
المعامل النوع الوصف
player Player اللاعب الذي انضم

OnPlayerDeath

يتم استدعاؤه عندما يموت لاعب

EventManager.OnPlayerDeath += (player, killer, damageType) =>
{
    if (killer != null)
        killer.SendMessage($"لقد قتلت {player.Nickname}!");
};

أحداث العناصر

OnItemPickup

يتم استدعاؤه عندما يلتقط لاعب عنصرًا

EventManager.OnItemPickup += (player, item) =>
{
    if (item.Type == ItemType.SCP268)
        player.SendMessage("احذر! هذا العنصر خطير");
};
