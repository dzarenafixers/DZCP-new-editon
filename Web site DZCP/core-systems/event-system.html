نظام الأحداث في DZCP

المقدمة

يسمح نظام الأحداث بالتفاعل مع جميع أحداث اللعبة عبر مشغلات (handlers) سهلة الاستخدام.

الأحداث المتاحة

حدث انضمام اللاعب

يتم تشغيله عندما ينضم لاعب جديد للسيرفر

EventManager.OnPlayerJoin += player =>
{
    player.SendMessage("مرحباً بك!");
};

حدث استخدام العنصر

يتم تشغيله عند استخدام أي عنصر في اللعبة


                        EventManager.OnItemUse += (player, item) =>
   {
        if (item.Type == ItemType.Medkit)
         player.Heal(50);
};

أفضل الممارسات

  • استخدم أحداثًا محددة بدلاً من التحقق المستمر
  • ألغِ تسجيل الأحداث عند تعطيل البلغن
