كيف تساهم في مشروع DZCP

طرق المساهمة

1. الإبلاغ عن الأخطاء

يمكنك فتح مشكلة (Issue) على GitHub للإبلاغ عن أي أخطاء أو مشاكل تواجهها.

صفحة المشاكل

2. تطوير ميزات جديدة

اختر ميزة من قائمة الميزات المطلوبة وقم بتطويرها:

# خطوات المساهمة:
1. Fork المشروع
2. إنشاء فرع جديد (git checkout -b feature/new-feature)
3. إجراء التعديلات
4. رفع التعديلات (git push origin feature/new-feature)
5. فتح طلب دمج (Pull Request)

إرشادات التطوير

  • التزم بأسلوب كتابة الكود المتبع في المشروع
  • أكتب توثيقًا واضحًا لأي كود تضيفه
  • أضف اختبارات لوظائفك الجديدة
  • استخدم أسماء توضيحية للمتغيرات والدوال
